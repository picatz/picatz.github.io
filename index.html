CODE & PROJECTS



Honey Cat

Command Line Application

#!/usr/bin/env bash

Honey Cat is a simple, easy to use honey pot that sets up what looks like is a service, but is really a honey pot on a user specified port. It is built with with BASH, Net Cat. Lolcat support is also a thing because people need it, obviously.

Red Honey Cat

Command Line Application

#!/usr/bin/env ruby

Red Honey Cat is a simple, easy to use honey pot that sets up what looks like is a service, but is really a honey pot on a user specified port. Originally written in BASH, Red Honey Cat is the Ruby variant I wrote for YoloCon 2016. Rainbow support is also a thing because people need it, obviously.


Wordpress Captcha Cracker

Command Line Application

#!/usr/bin/env ruby

A simple tool that will bypass the Wordpress security plugin AIOWPS ( All In One Wordpress Security & Firewall ) login captcha to act as a proof of concept that word mapping twenty numbers and then even base64 encoding the answer into the login page html is a bad idea for implementing a captcha to stop automation.

Rektcha

Command Line Application

#!/usr/bin/env bash

Rektcha is a simple captcha solving tool that analyzes a given image and solves the captcha for a captcha I found being used online. Supports different useful functions including visuals and custom output for feeding into other workflows.

Comcast-Cli

Browser Bot

#!/usr/bin/env ruby

A browser bot with a command line application to automate comcast router admin panel tasks using default credenetials or custom crednetials. Allows a neat little option to set a new password once you're in. Supports other options like rainbows as well.

Tito Bot

Slack Bot

#!/usr/bin/env ruby

A simple slack chat bot to give your team a little bit of Tito love and inspiration. Written in Ruby and updated recently to includes and easy to use command line interface to start the bot including, of course, a rainbow option.
